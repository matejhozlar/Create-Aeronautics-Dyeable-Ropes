## Version 1.2.0

### Added
- Added compatibility with the Dye Depot mod: dyeable rope couplings can now be crafted in all 16 additional Dye Depot colors. The extra recipes and tag entries only load when Dye Depot is installed, so a vanilla-only setup is unaffected.

### Fixed
- Fixed rope color texture generation incorrectly falling back to hardcoded colors when Dye Depot's extended color list was present, which caused mod-added colors to silently use wrong RGB values.
